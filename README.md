二、战斗对局

（3）枪械相关，制作多样化的武器 --基本全部实现，制作了手枪、冲锋枪、狙击枪等多种枪支

（4）命中规则 --未实现，直接采用角色物理资产

（5）怪物相关 ，包括怪物寻路索敌、优先攻击NPC，被玩家攻击后吸引仇恨，仇恨消失继续攻击NPC --全部实现
         近战、远程怪物 --全部实现
         
         拥有技能的BOSS --未实现
